01 / DOCTRINEThe five rules.

Five rules for the cabin.

A signature interface earns its strangeness. These are the ones I'm holding the rest of the design against. Break one and we slide back into dashboard.

RULE 01

Composition,
not layout.

Every screen has a focal mass, a counter-mass, and air between them. Never a uniform 12-col grid; always weighted, off-axis, intentional.

RULE 02

Depth is physical.

Three depth tiers, not nine: the void, the deck, the panels. Glass is structural, not decorative. Every shadow has a light source. No floating cards.

RULE 03

One red, used like ink.

Mars red is reserved for the live thing — the running session, the active task, the unread document. It is not a brand color. It is a status.

RULE 04

Type does the theatre.

Display serif sets the mood. Monospace metadata grounds it. Body in Inter does the working. Numerals live big — readable from across the room.

02 / NAVIGATIONThe orbit, not the rail.

Navigation as orbit.

The hub replaces the icon sidebar with a radial — a single living glyph at the center, six modules orbiting, the active one tightening to it. The page composes around the orbit, not around the corners.
